# 📚 PUCRS - Disciplina de Construçao de Compiladores (2026/1)

Este espaço será utilizado para a disponibilização de materiais teóricos, exemplos práticos de implementação e enunciados de exercícios ao longo do primeiro semestre de 2026.

## 🎯 Objetivos do Repositório

* Fornecer o código-fonte dos exemplos discutidos em aula.
* Centralizar os exercícios práticos e roteiros de laboratório.
* Servir como base para o desenvolvimento dos exercícios práticos do semestre.
---

## 🛠️ Tecnologias e Ferramentas

Para acompanhar as atividades deste repositório, recomenda-se o uso do **codespaces**. Com a configuração default e os códigos (jflex e byaccj) disponibilizados.


---

## 📂 Organização do Conteúdo

O repositório está organizado por fases do processo de compilação:

| Pasta | Descrição |
